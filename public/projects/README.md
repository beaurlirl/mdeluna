# Project Images

Place project image folders here. Each project should have its own folder named
to match the project `id` in `src/data/projects.js`.

Expected structure:

```
public/projects/
  <project-id>/
    cover.jpg
    1.jpg
    2.jpg
    3.jpg
```

Notes:
- `cover.jpg` is used for the project card and hero image.
- `1.jpg`, `2.jpg`, etc. are used for the gallery on the detail page.
- Use JPG or PNG; keep filenames lowercase.
